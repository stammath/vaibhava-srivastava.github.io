Light up! Denver

By Glenna Xie

This project maps the locations of every traffic signals in the City of Denver, and randomly generates the light-switching effect every five seconds. By parsing the GeoJSON data of the traffic signals in relation to each other, this visualization looks at how the city is connected through this nonstop flow produced by the transportation infrastructure.

This project is created using the data from the Denver Open Data Catalog and the JavaScript library d3.js.
